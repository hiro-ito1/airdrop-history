## 💬 メモ

自由記入欄：  
リネアは当初の期待値より少なかったが、AVNTで大きく上振れ。  
Quack AI は無料タスク＋AI実験としては驚くほどのリターン。  
Hyperliquid・DRIFT・Orderly などのトレード系はやはり強力。  
MONAD は、**「テストネットを一生懸命した人が報われない」**という意味で、今後のエアドロ文化を変える可能性あり。
ただ、ハイリキをFARCASTERやっていた人にエアドロがあった。資金を持っている人、暗号通貨に精通した人に絞った配り方をした傾向は見逃せない。
FARCASTERのミニアプリは、SCAMプロジェクト、プロジェクトは正しく運営されていても、軌道にのらない。ほとんどがそのようになっていそうだ。
通常の暗号通貨プロジェクトも小規模会社、FARCASTERは個人商店である。

## 💡 期待値・狙い目メモ

トレード系エアドロは、総じて **期待値が非常に高い。**  
特に、チェーン全体でのエコシステム構築が進んでいる  
**ハイパーリキッド（Hyperliquid）** や  
**BASEチェーン公認のアバンティス（Avantis）** は  
凄まじく高額エアドロとなった。

一方、タスク型・無料参加型は安定して「安全枠」。  
AI連携・ガバナンス系（例：Quack AI）は次世代の上振れ候補。

## 🧭 MONADエアドロ（2025年11月24日）の意味

- **テストネットタスク勢 → 報酬なし（ゼロ）**  
- **ハイリキ・Farcasterなどでの実活動ユーザー → 報酬あり**

MONADは、  
**タスク消化勢ではなく「本当に暗号通貨を使っているユーザー」を優遇する**  
という新しい姿勢を見せた。

優遇されたと思われる層：

- 実際に資金を動かしているユーザー  
- コミュニティに常時参加している暗号ネイティブ勢  
- Farcaster のような Web3 ソーシャル利用者  
- ハイリキなどで取引しているオンチェーンアクティブ層  

これが成功すると、  
**エアドロ ≠ タスク作業  
エアドロ ＝ 普段のオンチェーン活動の成果**  
へと大きく流れが変わる可能性がある。

## 🛡️ スキャム対策メモ（重要）

エアドロ界は詐欺が非常に多い。  
最低限これだけは覚えておく：

### ❌ 99% 詐欺  
- DMでの「当選通知」  
- メールでの「あなたにエアドロが配布されます」  

### 🔍 調べる時の正しい方法  
1. **正規Discordを見る**  
2. General / Announcements に  
   「DMが来たけど本物？」という投稿があることが多い  
3. 運営が  
   **『DMは送りません』**  
   と明記していることが多い  

### ☠️ よくある詐欺の誘導  
- 「Claimはこちら」  
- 「ウォレットを接続して本人確認」  
- 「エラーが出ています。サポートに連絡してください」  

→ *全部フィッシング。*
