## 💡 期待値・狙い目メモ

トレード系エアドロは、総じて **期待値が非常に高い。**  
特に、チェーン全体でのエコシステム構築が進んでいる  
**ハイパーリキッド（Hyperliquid）** や  
**BASEチェーン公認となったアバンティス（Avantis）** は  
凄まじく高額エアドロとなった。

一方で、**無料タスク系のエアドロ** は  
期待値はそれほど高くないが、**損はしない安全枠** として人気がある。
